How to help? 💕

The panda-lang organization is an open source initiative.
There are 4 main ways to support this project.

For everyone

  • Star Panda related projects on GitHub to help us reach a wider audience
  • Create some projects on GitHub to increase the popularity and library database

For involved

  • Join the developers team and develop projects associated with Panda organization
  • Consider a donation to help us in financial aspect of infrastucture

Your donation will fund Panda development
